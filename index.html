Counter Intelligence

Generated Content with CSS-Counters

Counter Intelligence is a preview tool for pseudo content generated by CSS Counters (spec).

Interesting Findings

  • Lower Roman range is 3999
  • Hebrew range is 999999
  • Lower Roman doesn't support negative numbrers
  • Hebrew doesn't support same Zero for all browsers
  • Arabic-Indic doesn't support negative numbers
  • Arabic-Indic support large numbers

Query With URL Params

  • offset Resets the CSS Counter to a starting point
  • range Amount of elements to render and count
  • language The Language Type used by the CSS Counter
